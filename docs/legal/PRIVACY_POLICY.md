## 4. Data Storage and Security

### 4.1 Local Storage (On Your Device)

**Chat Histories:**
- Stored locally using AsyncStorage
- Encrypted by your device's OS
- Not backed up to cloud
- Deleted when you uninstall the app

**Estimated Storage:**
- ~10KB per 100 messages
- Maximum ~10MB for extensive use

### 4.2 Cloud Storage

**Firebase (Google):**
- Character images and profiles
- App configuration data
- Anonymous usage analytics
- No personal chat content stored

**OpenRouter AI:**
- Temporary: Messages sent to generate AI responses
- Not stored permanently
- Used only during conversation
- Deleted after response generated

**Sentry (Crash Reporting):**
- Error logs and stack traces
- Device information
- No chat content
- Anonymized data only

### 4.3 Security Measures

- ✅ HTTPS encryption for all network requests
- ✅ API keys filtered from error logs
- ✅ Local data encrypted by OS
- ✅ No cloud backup of chats
- ✅ Regular security updates
