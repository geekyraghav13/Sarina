# Background Videos

This directory should contain the background video files for the app.

## Required Videos

Place the following .mp4 video files in this directory:

- `default.mp4` - Default background video (required)
- `flirty.mp4` - Flirty tone background (optional)
- `cute.mp4` - Cute tone background (optional)
- `romantic.mp4` - Romantic mode background (optional)
- `playful.mp4` - Playful tone background (optional)
- `mysterious.mp4` - Mysterious tone background (optional)

## Video Specifications

- Format: MP4 (H.264)
- Resolution: 1080x1920 (9:16 aspect ratio for mobile)
- Duration: 10-30 seconds (should loop seamlessly)
- Size: Keep under 10MB per video for optimal performance
- Style: Cinematic, atmospheric, with soft lighting

## Temporary Solution

For development, you can use any MP4 video file and name it `default.mp4`.
You can find free stock videos at:
- Pexels (https://www.pexels.com/videos/)
- Pixabay (https://pixabay.com/videos/)
- Coverr (https://coverr.co/)

Search for keywords like:
- "ambient background"
- "atmospheric"
- "bokeh lights"
- "gradient motion"
- "cinematic"
