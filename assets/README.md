# Assets Directory

## Required Assets

The following image assets need to be added:

1. **icon.png** (1024x1024) - App icon
2. **adaptive-icon.png** (1024x1024) - Android adaptive icon
3. **splash.png** (1242x2436) - Splash screen image
4. **favicon.png** (48x48) - Web favicon

## Temporary Solution

For development, you can create simple placeholder images or use the default Expo assets.

To generate proper app icons and splash screens, you can use:
- https://www.appicon.co/
- https://icon.kitchen/
- Figma or other design tools

## Note

Since these are image files, they cannot be created through code.
Please add them manually or use design tools to create them.
